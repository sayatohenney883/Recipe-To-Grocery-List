# Recipe-To-Grocery-List

As someone who loves food, I struggle to efficiently create a concrete shopping list based on the recipes I want to try for the week.
I want to create a program that helps users create a shopping list with the ingredients required for the recipes. 
The program's purpose is to help develop my coding, project creation, and critical thinking skills while creating a useful program for users to utilize.

On top of the personal gain in terms of experience, my father directly asked for a project of this sort to be completed.
Having the capability to assist with my father's requests through coding makes me feel so eager to learn more about the world of computer science.
This project will demonstrate the passion that I have when it comes to creating an original idea that benefits not just myself, but others who choose to use this program. 

If any improvements can be given, please give them. I love to continue to enhance my skills and capabilities!

Thank you!
